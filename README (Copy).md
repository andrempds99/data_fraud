# datascience-takehome
Please take at most 8-10 hours (of non-training time) for this task. The goal is to train a classifier and predictions for this data to be specific:

* Check data quality
* Expand features 
* Control class imbalance
* Train classifiers
* Plot Metrics
* Detect overfit and mitigate it
* Optimise for different recalls (say 30% - 60%)

We would like to check your toolkit and how you tackle problems. Some of the description is intentially left vague to see how you deal with it. However, if you feel that something is too unclear or wrong please reach out to us and we will try to assist.
